Différence entre les CPU et GPU

💡 Différences entre CPU, GPU et FPGA :

  • CPU (Processeur classique) : Conçu pour exécuter des tâches générales.
  • GPU (Processeur graphique) : Optimisé pour le calcul parallèle (idéal pour l’IA et le rendu graphique).
  • FPGA : Circuits reconfigurables capables d’exécuter des tâches spécialisées avec une efficacité maximale.

📌 Pourquoi utilise-t-on les FPGA ?

Les FPGA sont présents dans de nombreux domaines, notamment :

  • Voitures autonomes : Traitement rapide des données des capteurs.
  • Cryptomonnaies : Accélération des calculs pour le minage (ex. Bitcoin).
  • Télécommunications : Optimisation des signaux et traitement réseau.

Projet - Un mini-processeur en FPGA

⚙️ 1. Conception d’une ALU simple

Une ALU (Arithmetic Logic Unit) est le cœur d’un processeur. Elle exécute les opérations mathématiques et logiques.

✅ Objectifs :

  • Construire une ALU simplifiée capable d’effectuer une addition ET une comparaison.
  • Découvrir comment un processeur exécute des opérations en modifiant ses entrées.

🛠️ Étape 1 : Création des opérations de base

Les élèves doivent ajouter deux modules principaux :

  • Un additionneur 2 bits (utilisant des portes logiques OU et ET).
  • Un comparateur (A > B, A < B, A = B).

🛠️ Étape 2 : Sélection des opérations

Ajout d’un multiplexeur qui sélectionne :

  • Mode 0 : L'ALU effectue une addition.
  • Mode 1 : L'ALU effectue une comparaison.

# Exemple de code pour une ALU en langage VHDL (FPGA)
process(A, B, mode)
begin
  if mode = '0' then
    S <= A + B; -- Addition
  else
    if A > B then
      S <= "01"; -- A > B
    elsif A = B then
      S <= "00"; -- A = B
    else
      S <= "10"; -- A < B
    end if;
  end if;
end process;

🎯 Défi : Ajout d’une opération XOR ou inversion des bits

Pour aller plus loin, les élèves peuvent ajouter une opération XOR en utilisant des portes logiques XOR.
