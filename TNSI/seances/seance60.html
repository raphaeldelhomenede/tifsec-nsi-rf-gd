Séance 60 : Test sur les Bases de Données et Python

Exercice 1 : Modélisation et requêtes SQL (8 points)


Une entreprise gère une base de données contenant des informations sur ses employés et leurs projets. La base de données comporte les tables suivantes :

  • Employe(id_employe, nom, prenom, date_naissance, salaire)
  • Projet(id_projet, nom_projet, budget)
  • Travaille(id_employe, id_projet, heures_travaillees)
  1. (2 pts) Ecrire une requête SQL permettant d'afficher le nom et le prénom de tous les employés qui travaillent sur au moins un projet.
  2. (2 pts) Ecrire une requête SQL pour afficher le nom des projets dont le budget est supérieur à 50 000.
  3. (2 pts) Ecrire une requête SQL permettant de calculer le nombre total d'heures travaillées par employé.
  4. (2 pts) Quelle clé étrangère est présente dans la table Travaille ? Justifiez votre réponse.
  5. (2 pts) Écrire une requête SQL permettant d'afficher les employés qui travaillent sur plus d’un projet.
