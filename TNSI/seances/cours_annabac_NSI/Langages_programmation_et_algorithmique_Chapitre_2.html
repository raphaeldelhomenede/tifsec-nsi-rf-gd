🔹 Qu’est-ce que la récursivité ?

Une fonction est récursive lorsqu’elle s’appelle elle-même. C’est une technique puissante pour résoudre des problèmes définis de manière récursive.

Une fonction récursive possède deux parties :

  • Cas de base (ou cas d'arrêt) : condition pour arrêter l'appel récursif.
  • Appel récursif : la fonction s'appelle avec un cas plus simple.

📌 Exemple 1 : Factorielle

def factorielle(n):
    if n == 0:
        return 1  # Cas de base
    else:
        return n * factorielle(n - 1)  # Appel récursif

print(factorielle(5))  # Affiche 120

Ici, factorielle(5) appelle factorielle(4), qui appelle factorielle(3), etc., jusqu'à factorielle(0).

🔄 Comparaison récursivité / itération

La récursivité peut souvent être remplacée par une boucle.

Itératif :

def factorielle_iter(n):
    resultat = 1
    for i in range(1, n + 1):
        resultat *= i
    return resultat

Récursif :

def factorielle_rec(n):
    if n == 0:
        return 1
    return n * factorielle_rec(n - 1)

Les deux donnent le même résultat, mais l’itératif est généralement plus rapide et évite les erreurs de dépassement de pile (stack overflow).

🧪 Autres exemples

Exemple 2 : Somme des entiers

def somme(n):
    if n == 0:
        return 0
    return n + somme(n - 1)

print(somme(4))  # Affiche 10

Exemple 3 : Suite de Fibonacci

def fibonacci(n):
    if n == 0:
        return 0
    elif n == 1:
        return 1
    return fibonacci(n - 1) + fibonacci(n - 2)

print(fibonacci(6))  # Affiche 8

⚠️ Cette version est lente pour de grandes valeurs de n car elle répète beaucoup de calculs.

📉 Complexité et récursivité

  • La récursivité peut simplifier le code
  • Mais elle peut être inefficace si mal utilisée
  • Utiliser des techniques comme la mémoïsation ou passer à une version itérative

🧪 Tests et vérifications

def carre(x):
    return x * x

def test_carre():
    assert carre(2) == 4
    assert carre(-3) == 9

print("Tests passés !")
test_carre()

Il est aussi possible d’écrire des tests pour les fonctions récursives comme pour les autres.
