Structures de Données
Chapitre 3 : Graphes
1. Introduction aux graphes
Un graphe est une structure de données permettant de modéliser des relations entre des objets. Il est constitué de sommets (ou nœuds) et d'arêtes (ou arcs).
  • Sommets : représentent les objets
  • Arêtes : représentent les connexions entre les objets
2. Types de graphes
  • Graphe orienté : les arêtes ont une direction (ex : A → B)
  • Graphe non orienté : les arêtes n'ont pas de direction (A ↔ B)
  • Graphe pondéré : les arêtes ont un poids (ex : coût, distance)
  • Graphe non pondéré : les arêtes sont toutes égales
3. Représentation des graphes
  • Liste d'adjacence : chaque sommet a une liste de ses voisins
  • Matrice d'adjacence : tableau 2D indiquant la présence ou le poids d'une arête
Exemple : Liste d'adjacence
A : B, C
B : A, D
C : A
D : B
Exemple : Matrice d'adjacence (non orienté)
    A B C D
A [ 0 1 1 0 ]
B [ 1 0 0 1 ]
C [ 1 0 0 0 ]
D [ 0 1 0 0 ]
4. Parcours de graphe
  • Parcours en largeur (BFS) : explore les voisins d'abord
  • Parcours en profondeur (DFS) : explore un chemin jusqu'au bout avant de revenir
Exemple de BFS (A, B, C, D)
Exemple de DFS (A, B, D, C)
5. Implémentation d'un graphe non orienté en Python
class Graphe: def __init__(self): self.adjacence = {} def ajouter_sommet(self, sommet): if sommet not in self.adjacence: self.adjacence[sommet] = [] def ajouter_arete(self, sommet1, sommet2): self.adjacence[sommet1].append(sommet2) self.adjacence[sommet2].append(sommet1) # Exemple mon_graphe = Graphe() mon_graphe.ajouter_sommet("A") mon_graphe.ajouter_sommet("B") mon_graphe.ajouter_sommet("C") mon_graphe.ajouter_arete("A", "B") mon_graphe.ajouter_arete("A", "C")
6. Applications des graphes
  • Réseaux sociaux (utilisateurs connectés)
  • Plans de villes et GPS
  • Réseaux informatiques
  • Organisation de dépendances (tâches, compilateurs)
  • Web : structure des liens entre pages
7. Résumé du chapitre
  • Un graphe est formé de sommets et d'arêtes
  • On peut le représenter avec une liste ou une matrice d'adjacence
  • Deux grands types de parcours : BFS et DFS
  • Utilisé dans de nombreux domaines pratiques
