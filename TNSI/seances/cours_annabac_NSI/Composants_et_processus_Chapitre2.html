Chapitre 2 : Protocoles de Routage et Sécurisation des Communications
1. Introduction
Dans ce chapitre, nous explorons les protocoles de routage utilisés pour acheminer les données dans un réseau ainsi que les mécanismes de sécurisation des communications.
2. Protocoles de routage
Le routage consiste à choisir le meilleur chemin pour envoyer les données d'une source à une destination sur un réseau.
  • Routage statique : chemins configurés manuellement, utilisé dans les petits réseaux.
  • Routage dynamique : les routeurs ajustent automatiquement leurs routes avec des protocoles comme :
    • RIP (Routing Information Protocol) – protocole à vecteur de distance.
    • OSPF (Open Shortest Path First) – protocole à état de lien, plus rapide et adaptatif.
    • BGP (Border Gateway Protocol) – utilisé sur Internet pour le routage entre organisations.

4. Sécurisation des communications
Il est essentiel de protéger les données pendant leur transfert. Pour cela, on utilise :
  • Chiffrement : transforme les données pour qu'elles soient inaccessibles sans clé.
  • Protocoles de sécurité :
    • HTTPS : HTTP + TLS pour des connexions web sécurisées.
    • SSL/TLS : protège les communications sur Internet.
    • VPN : crée un tunnel chiffré entre l'utilisateur et le réseau distant.
    • IPSec : sécurise les communications à l’aide de l’IP.
5. Menaces et attaques courantes
  • Interception (sniffing) : capture des paquets non chiffrés.
  • Attaque Man-in-the-Middle : interception et modification des messages entre deux parties.
  • Usurpation d'identité (spoofing) : se faire passer pour un autre appareil.
La mise en place de pare-feux, IDS/IPS et de politiques de sécurité strictes permet de limiter ces risques.
6. Récapitulatif
  • Le routage permet d'acheminer les paquets sur un réseau.
  • Les protocoles comme RIP, OSPF, BGP ont des usages différents.
  • La sécurisation des communications repose sur le chiffrement et des protocoles spécifiques.
  • Des mesures de prévention sont indispensables pour contrer les attaques réseau.
